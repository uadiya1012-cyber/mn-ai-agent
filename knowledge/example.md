# mn-ai-agent — жишээ мэдлэгийн баримт

Энэ файлыг RAG туршихад ашиглана. `knowledge/` хавтас руу өөрийн `.md`, `.txt` файлуудаа хуулна.

## Төслийн тухай

**mn-ai-agent** нь Монгол болон Англи хэл дэмжсэн AI туслагч:
- Claude API — контент, tool use
- Ollama — локал код, мэдээ/пост үүсгэгч
- RAG — энэ хавтасны баримтаас хайлт хийж хариултанд ашиглана

## RAG ашиглах

```bash
ollama pull nomic-embed-text
uv run python main.py rag index
uv run python main.py rag search "mn-ai-agent гэж юу вэ"
```

Agent чат дээр: «Манай төслийн RAG хэрхэн ажилладаг вэ?» гэж асууна.
